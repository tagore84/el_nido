# Workflows

Aquí residen los ficheros JSON que definen los workflows de n8n.

## Estructura

- `agenda/`: Gestión de calendarios y eventos.
- `coco/`: Todo lo relacionado con la mascota.
- `compra/`: Listas de compras y gestión de inventario.

## Nota Importante

Estos ficheros son generados por el script de exportación. **No editar manualmente** el JSON a menos que se sepa exactamente qué se está haciendo. Utilizar n8n para editar visualmente y luego exportar.
